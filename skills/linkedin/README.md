# LinkedIn

Stable skills for LinkedIn authentication, research, drafting, engagement, and
other LinkedIn-specific workflows.

Stable skills in this bucket are included in generated manifests.

- **[linkedin-engagement](./linkedin-engagement/SKILL.md)** - Gather LinkedIn
  posts, classify engagement opportunities, draft comments, and require explicit
  approval before posting.
- **[linkedin-login](./linkedin-login/SKILL.md)** - Capture LinkedIn
  authentication cookies and save them to a project `.env` file.
